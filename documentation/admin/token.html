Generate tokens

Here's a guide to generate tokens through the admin api, this is the data that needs to be post'd :


name : admin_token -> value : your admin token

name : program_key -> value : your program key

name : token_amount -> value : the token amount

name : token_days -> value : the amount of days of the token(s)

name : token_level -> value : the token(s) level

OPTIONALS =>

name : token_type -> value : the int identifier of the token type ( 1 -> normal, 2 -> big normal, 3 -> uppercase guid, 4 -> custom mask ) ( by default 1 )

name : custom_mask -> value : the custom mask of the token ( X => random char ) ( will only be used if the token type is 4 )

complete api link for generating tokens : https://cauth.me/api/admin_handler.php?type=gen_token
