Fetch single user

Here's a guide to fetch a single user, this is the data that needs to be post'd :


name : admin_token -> value : your admin token

name : program_key -> value : the program key

name : username -> value : the username you want to fetch

complete api link to fetch a single user : https://cauth.me/api/admin_handler.php?type=fetch_single_user
